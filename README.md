# Portfolio Website

This is a personal portfolio website built with PHP, HTML, CSS, and JavaScript. It showcases projects, skills, and provides a contact form for visitors to reach out.

## Features
- **Home Page** (`index.php`): Introduction and highlights.
- **About Page** (`about.php`): Information about yourself.
- **Contact Page** (`contact.php`): Contact form with email functionality.
- **Header and Footer** (`header.php`, `footer.php`): Consistent navigation and branding.
- **Custom Styles** (`style.css`, `header.css`, `contact.css`): Modern, responsive design.
- **JavaScript Enhancements** (`script.js`): Interactive features.
- **Images** (`images/`): Portfolio and profile images.

## Getting Started
1. **Requirements:**
   - PHP 7.x or higher
   - Web server (e.g., XAMPP, Apache)
2. **Setup:**
   - Clone or copy the repository to your web server directory (e.g., `htdocs` for XAMPP).
   - Start your web server and navigate to the project folder in your browser.
3. **Configuration:**
   - Update contact email in `contact.php` (`$to` variable) to your own email address.
   - Add your own content, images, and project details.

## Customization
- Edit `about.php` and `index.php` to add your bio and projects.
- Update styles in `style.css`, `header.css`, and `contact.css` for your preferred look.
- Add images to the `images/` folder and reference them in your pages.
